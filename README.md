# 🖱️ HoverMind: The AI-Powered Windows Cursor

HoverMind turns your standard Windows mouse pointer into a context-aware AI assistant. Simply hold down `Alt + Shift` and hover over any image, block of code, foreign text, or complex UI element to instantly get an AI-generated explanation of what you are looking at.

## ✨ Features
- **Look at Anything:** Uses state-of-the-art Vision-Language Models to "see" your screen.
- **Unobtrusive UI:** A beautiful, frosted-glass floating tooltip that follows your cursor and disappears when you release the hotkey.
- **DPI-Aware:** Perfectly captures screen snippets regardless of your monitor setup or Windows scaling.
- **Multi-Modal:** Can translate text, explain confusing UI icons, identify plants/objects in photos, or summarize code snippets right in your IDE.
